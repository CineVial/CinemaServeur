Service RESTFUL pour le projet "Cinéma", version web hébergée à l'adresse api.cinema-epul.tk.

Services accessibles :

  • liste des entités :

    • /acteurs
    • /categories
    • /films
    • /personnages
    • /realisateurs
  • accès à une entité :

    • /acteur?a_id=X
    • /categorie?c_id=X
    • /film?f_id=X
    • /personnage?a_id=X&f_id=X
    • /realisateur?r_id=X
